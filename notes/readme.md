02 - Illustrative Bland-Altman plot comparing two simulated heart-rate sensors. The PPG-like sensor showed a small planted negative bias relative to the ECG-like reference sensor, with a mean difference of -1.57 bpm and limits of agreement from -6.59 to 3.45 bpm. This example demonstrates how agreement analysis addresses sensor interchangeability rather than predictor-outcome association.

03 - Illustrative mixed-effects model of resting heart rate across age. Each row represents a simulated session-level summary from one infant at one age. Thin lines show example infant-level trajectories, while fitted lines show the estimated population trajectory by group. The model recovered the planted age-related decline in resting heart rate and accounted for repeated sessions within infants through infant-level random effects. This example demonstrates how mixed-effects models handle clustered, unbalanced naturalistic infant data by estimating population-level developmental change while allowing infants to differ in baseline physiology.

04 - In the illustrative prediction example, row-shuffled cross-validation produced higher and less variable balanced accuracy than infant-blocked cross-validation. Because row-shuffled splits allow sessions from the same infant to appear in both training and test folds, they can inflate apparent model performance by exploiting subject-specific structure. Infant-blocked validation provides a more stringent estimate of generalization to unseen infants.
    * Dashed line indicates chance-level balanced accuracy for a three-class prediction task.

05 - In the illustrative state-space example, a latent respiratory-rate trajectory was estimated from noisy and intermittently missing sensor observations. The raw sensor measurements deviated from the simulated latent respiratory rate by approximately 1.97 breaths/min, whereas the Kalman/state-space estimate reduced this error to approximately 0.83 breaths/min at observed time points. Across the full recording, including dropout periods, the state-space estimate maintained an error of approximately 0.88 breaths/min. This example demonstrates how state-space models can combine prior estimates with incoming measurements to track a continuously evolving physiological state, while remaining robust to sensor noise and short missing-data intervals.
