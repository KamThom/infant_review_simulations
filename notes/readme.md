02 - Illustrative residual analysis showing the effect of an omitted contextual variable. In the context-omitted model, residuals retain a systematic bump, indicating that the model fails to capture structure in the simulated data. After the context variable is included, residuals flatten toward zero and the estimated social-exposure effect is reduced toward its planted value. This example demonstrates how residual diagnostics can reveal unmeasured or omitted confounding in naturalistic infant sensing analyses.

03 - Illustrative mixed-effects model of resting heart rate across age. Each row represents a simulated session-level summary from one infant at one age. Thin lines show example infant-level trajectories, while fitted lines show the estimated population trajectory by group. The model recovered the planted age-related decline in resting heart rate and accounted for repeated sessions within infants through infant-level random effects. This example demonstrates how mixed-effects models handle clustered, unbalanced naturalistic infant data by estimating population-level developmental change while allowing infants to differ in baseline physiology.

04 - In the illustrative leakage example, a model trained only on deployable physiological and contextual features performed near chance. When a leaked recording-context feature was included, apparent performance rose to perfect classification. However, when that shortcut was unavailable at deployment, performance collapsed back toward chance. This demonstrates how predictive models trained on repeated naturalistic infant data can appear highly accurate by exploiting non-generalizable metadata, batch, or recording-context artifacts rather than learning a meaningful developmental signal.

05 - In the illustrative state-space example, a latent respiratory-rate trajectory was estimated from noisy and intermittently missing sensor observations. The raw sensor measurements deviated from the simulated latent respiratory rate by approximately 1.97 breaths/min, whereas the Kalman/state-space estimate reduced this error to approximately 0.83 breaths/min at observed time points. Across the full recording, including dropout periods, the state-space estimate maintained an error of approximately 0.88 breaths/min. This example demonstrates how state-space models can combine prior estimates with incoming measurements to track a continuously evolving physiological state, while remaining robust to sensor noise and short missing-data intervals.

06 - Illustrative Bland-Altman plot comparing two simulated heart-rate sensors. The PPG-like sensor showed a small planted negative bias relative to the ECG-like reference sensor, with a mean difference of -1.57 bpm and limits of agreement from -6.59 to 3.45 bpm. This example demonstrates how agreement analysis addresses sensor interchangeability rather than predictor-outcome association.
