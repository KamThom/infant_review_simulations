02 - Illustrative residual analysis showing the effect of an omitted contextual variable. In the context-omitted model, residuals retain a systematic bump, indicating that the model fails to capture structure in the simulated data. After the context variable is included, residuals flatten toward zero and the estimated social-exposure effect is reduced from 0.475 to 0.203, closer to the planted value of 0.22. This example demonstrates how residual diagnostics can reveal unmeasured or omitted confounding in naturalistic infant sensing analyses.

03 - Illustrative mixed-effects model of resting heart rate across age. Each row represents a simulated session-level summary from one infant at one age. Thin lines show example infant-level trajectories, while fitted lines show the estimated population trajectory by group. The model recovered a reference-group age-related decline of -1.111 bpm/month after 3 months, with the comparison group starting 4.179 bpm higher and declining slightly less steeply. This example demonstrates how mixed-effects models handle clustered, unbalanced naturalistic infant data by estimating population-level developmental change while allowing infants to differ in baseline physiology.

04 - Illustrative automated infant-position labeling from simulated wearable windows. A classifier trained on movement intensity, posture angle, movement variability, vertical acceleration, and horizontal acceleration converted 10-second windows into held, supine, prone, and sitting labels. The held-out balanced accuracy was 0.923 across 180 windows, with most errors occurring between supine and prone windows. This example demonstrates how window-level sensor features can be transformed into a behavioral timeline while preserving enough diagnostic output to inspect failure modes.

05 - In the illustrative state-space example, a respiratory-rate trajectory was estimated from noisy and intermittently missing sensor observations. The raw sensor measurements deviated from the simulated latent respiratory rate by approximately 1.98 breaths/min, whereas the Kalman/state-space estimate reduced this error to approximately 0.68 breaths/min at observed time points. Across the full recording, including dropout periods, the state-space estimate maintained an error of approximately 0.67 breaths/min. The plot emphasizes the noisy observations and state-space estimate without showing the simulated latent truth line, because that line is diagnostic scaffolding rather than the communication target.

06 - Illustrative Bland-Altman plot comparing two simulated heart-rate sensors. The PPG-like sensor showed a small planted negative bias relative to the ECG-like reference sensor, with a mean difference of -1.57 bpm and limits of agreement from -6.59 to 3.45 bpm. This example demonstrates how agreement analysis addresses sensor interchangeability rather than predictor-outcome association.

07 - In the illustrative leakage example, a model trained only on deployable physiological and contextual features performed near chance, with mean balanced accuracy of 0.526. When a leaked recording-context feature was included, apparent performance rose to 1.000. However, when that shortcut was unavailable at deployment, performance collapsed back toward chance, with mean balanced accuracy of 0.512. This demonstrates how predictive models trained on repeated naturalistic infant data can appear highly accurate by exploiting non-generalizable metadata, batch, or recording-context artifacts rather than learning a meaningful developmental signal.

08 - Optional Fleming-style reference view of the simulated heart-rate trajectory. The same mixed-effects fitted group means are shown as step-style trajectories against approximate pediatric heart-rate reference anchors based on Fleming et al. 2011. This figure is intended as a visual alternative for comparing the simulated developmental pattern with a published reference-frame style, not as a formal centile model or empirical validation.
